# My dotfiles

## Installation
```bash
git clone https://github.com/artemgurzhii/dotfiles.git &&
cd dotfiles &&
./bootstrap.sh
```

## Notes
Don't forget to override some of the config files(like `.gitconfig`), which includes some my info
